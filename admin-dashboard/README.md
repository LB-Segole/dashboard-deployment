# Admin Dashboard

This is the admin dashboard for the VoiceAI platform.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Linting

```bash
npm run lint
```

## Formatting

```bash
npm run format
```

## Type Checking

```bash
npm run type-check
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```
